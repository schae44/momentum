# Momentum
Momentum is a personal website and offers functionalities such as a JS clock, a JS to-do list, geolocation retrieval and weather information display.

## Features

- [x] Clock
- [x] To Do List
- [x] Weather
- [x] Daily Quote
- [x] Daily Photo
- [x] Offline Support
